File Formats

JPEG

  • JPEG Join Photgraphic Experts Group --ideal for photographs and other continuous-tone images such as paintings adn grayscale images< /li>
  • lossy compression --reduces the file size by eliminating pixel information with each save
  • artifacts --blotches and noise appear at edges and in areas of flat color
  • lossy compression
  • can create artifacts

GIF

  • GIF Graphic Interchange Format --8 bits, no more than 256 colors
  • lossless --no pixel information is lost
  • run-length compression also called LZW compression

8-Bits or Less Color

  • color palette (color table or color map)
  • web-safe color palette --216 colors shared by Windows and Mac
  • Anti-aliasing --visual "smoothing" of diagonal edges and contrast endges via pixels of intermediate colors along boundary edges

Transparency

GIF has only 1 bit transparency, either fully transparent or fully opaque

Animation

GIFs can be animated

  • cinemagraph

PNG

Better option thatn GIF.

  • lossless compression
  • 8-bits indexed color as well as full 24-bit true color
  • From 1 to 8 bits of transparency
  • PNG Portable Network Graphics

SVG -vector format

  • SVG -- Scalable Vector Graphis

SVG do not lose quality when elanrged or reduced
