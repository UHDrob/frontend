4. File Formats

  • JPEG
    • lossy compression
    • can create artifacts
  • GIF
    • pronunciation controversy
    • GIF is an 8=bit or less format, meaning that it can contain no more than 256 colors
    • Not great for photos
  • PNG
    • lossless compression
    • 8-bit indexed color as well as full 24-bit true color
    • From 1 to 8 bits of Transparency (256 levels)
  • SVG
    • Very clear to enlarge images, no loss in quality
    • Supported in XML
  • TIF
    • multiple color depth, good for printing
  • WebP
    • support both lossy and lossless compression
