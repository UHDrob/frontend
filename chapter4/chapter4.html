      Attribute Selectors
      Selector Matches
      [] A specific attribute
      [=] A specific attribute with a specific value
      [~=] A specific attribute whose value matches at least one of the words in a space-delimited list of words.
      [^=] A specific attribute whose value begins with a specified value.
      [*=] A specific attribute whose value contains a substrings.
      [$=] A specific attributes whose value ends with a specified value.

      Selector Description
      a:link Select links that have not been visited.

    • 4.6.1 Background
      • Common background properties
      Property Description
      background A combined shorthand property that allows you to set multiple backgrounds values in one property
      background-attachment Specifies whether the background image scrolls with the document(default) or remains fixed. [fixed, scroll]
      background-color Sets the background color of the element
      background-image Specifies the background image for an element.
      background-position Specifies where on the element the background image will be placed. [bottom, center, left, right, 300px 50px]
      background-repeat Determines whther the background image will be repeated [repeat, repeat-x, repat-y, no-repeat]
      background-size This property lets you modify the size of the background image
    • 4.6.2 Borders
      • Border Properties
      Property Description
      border A combined shorthand property that allows you to set the style, width, and color of a border in one property [border-style border-width border-color]
      border-style Specifies the line type of the border [solid, dotted, dashed, double,groove, ridge, inset, and outset
      border-width The widht of the border in a unit (but not in percents)
      border-color The color of the border in acolor unit.
      border-radius The radius of a roudend corner.
      border-image The URL of an image to use as a border

    • Font Properties
    Property Description
    font A combined shorthand property that allows you to set the family, style, size, variant, and weigth in one property (minumum font size and font family)
    font-family Specifies the typeface/font (or generic font family) to use
    font-size The size of the font in one of the measurement units.
    font-style specifies whether italic, oblique, or normal
    font-variant Specifies either small-caps text or none
    font-weight Specifies either normal, bold, bolder, ligher, or a value between 100 and 900 in multiples of 100, where larger number represents weightier(i.e. bolder) text

      Text Properties
      Property Description
      letter-spacing Adjust the space between letters. Can be the value normal or a length unit
      line-height Specifies the space between baselines
      list-style-image Specifies the URL of an image to use as the marker for unordered lists. Set to none to remove markers
      list-style-type Selects the market type to use for ordered and unordered lists. Nonte to remove markers
      text-align Aligns the text horizontally in a container element in a similar way as a word processor. [left, right, center, justify]
      text-decoration Specifies whether the text will have lines below, through, or over it [none, underline, overline,line-through, blink]
      text-direction Specifies the direction of the text [ltr, rtl]
      text-indent Indents the first line of a paragraph by a specific amount
      text-shadow A new CSS3 property that can be used to add a drop shadow to a text
      text-transform Changes the capitalization of text. [none, capitalize, lowercase, uppercase]
      vertical-align Aligns the text vertically in a container element [top, bottom, middle]
      word-spacing Adjust the space between words. can be the value normal or a length unit
