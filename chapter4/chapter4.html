Algorithm to determine specificity of any given element

  1. value a: First count 1 if the declaratin is from a "style" attribute in the HMTL, 0 otherwise
  2. value b: Count the number of ID attributes in the selector
  3. value c: Count the number of class selectors, attribute selectors, and pseudo-classes in the selector
  4. value d: Count the number of elements names and pseudo-elements in the selector
  5. Finally, concatenate the four numbers a + b + c + d
