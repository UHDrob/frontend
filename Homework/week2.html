Describe the 4 layers in the four-layer network model

Application Layer: higher protocols that allow applicaitons to interact with transport layers.

The application layer is the higherst layer in the TCP/IP model. Here it interacts with the user. For instance: web browsers, email, chat programs and others. Protocols: HTTP, SSH, FTP, POP/IMAP/SMTP, and DNS.

Transport Layer: ensures transmissions arrive in order and without error.

It ensures that the applicaiton data is delivered to the receiving device in the correct order and without errors. Protocols: TCP and UDP

Internet Layer: establishes connection, routing, and addressing.

It assigns an IP address to devices, and it is reponsible for establishing the connection between two end hosts and determine where the traffic should be sent. Protocols: IPv4 and IPv6.

Link Layer: resposible for physical transmission of raw bits.

Here the data is converted into digital signals that can be transmitted over copper wires or fiber optics. Protocol: MAX.

What does it mean browser rendering? Browser caching?

Browser rendering is when the browser interprets all HTML markup, with the images and information, into a grid of pixels to display within the browser window.

Browser caching is saving the unchanged part of the website locally and only download what is new. The process speeds up the browser and reduce the network load.
