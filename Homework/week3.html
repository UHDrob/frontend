List the advantages of having semantic markup in an html document

Maintainability: semantic markup is easier to update and change

Performance: web pages performance faster improves when recognizes html tags.

Accessibility: modern browsers know how to access semantic tags for other audiences with disabilities, such as users with sight disabilities.

Search Engine Organization SEO looks for semantics or level or importance for content on the site so search engine crawlers can easily identify important content on the site.
